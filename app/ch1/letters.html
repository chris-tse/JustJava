Chars and Strings

In Java there are two main ways of storing text. Text can either be a single character, or a sequence of characters called strings.

When declaring a character in Java, we use the type char and surround the character with single quotes.

For several characters in a row, like whole words and sentences, a String type must be used, and the string should be wrapped in double quotes (note that String is capitalized).
