Now that we've covered ways to evaluate logical expressions, we can move on to actually incorporating them into our programs to make decisions.

Here is the general syntax of an if statement in Java:

if (condition) {
	// Your code here
}

When using an if statement, the program will first evaluate the expression inside the parentheses, which should return a boolean. If it evaluates as true the program will then execute the block of code inside the curly braces. If false, the program will skip the block of code completely.
