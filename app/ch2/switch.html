Switch Statement

Back to Table of Contents

While we can chain multiple if / else if blocks together to check for multiple outcomes of a condition as shown in a previous section, there is a way to do so with more concise code using the switch statement.

The syntax is as follows:

switch (condition) {
	case a:
		// if condition == a, do something
		break;
	case b:
		// else if condition == b, do something else
		break;
	default:
		// executes if condition was not a nor b	
}

The difference with a switch statement is that instead of checking true or false in the condition, you are checking whether condition matches a or b. You can have as many cases as needed. The switch statement can check byte, short, char, and int primitive types and their respective wrapper classes along with String types.

		System.out.println("Saturday");
