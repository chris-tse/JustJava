Ternary Operator

Back to Table of Contents

Now that we've covered how to make if statements more efficient using else, we can look at a more concise way of making decisions using something called the ternary operator.

The syntax of the ternary operator is as follows: 

(condition ? expr1 : expr2);

If the condition is true, the expression returns the value of expr1. Otherwise, it returns the value of expr2. For example, if you want to compare the value of two integers, and store the smaller value into min, you can do this:

int a = 3;
int b = 9;
int min = ( a < b ? a : b);

The condition statement is a < b, and if it is true, it will return a, otherwise it will return b.

Below are some examples that demonstrating some basic example using ternary operator 

// if (a+b) great than c, return a, else return b
int result = (a+b) > c ? a : b;
//if a is not equal to b, print a, otherwise print b
(a != b) ? System.out.print(a): System.out.print(b);
