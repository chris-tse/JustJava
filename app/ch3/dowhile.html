3.3 Do While Loops

Back to Table of Contents

The last type of loop we will be looking at is the do while loop, and the syntax is as follows:

do {
	// your code here
} while (condition); // note the semi-colon here

You will notice a difference between the do while and while loop, and it is that it evaluates the condition at the bottom of the loop instead of the top. Therefore, the statements within the do block are always executed at least once regardless of the condition.

A way to demonstrate this is to first take a look at the following while loop:

while (false) { // condition is false, does not enter loop
    System.out.println("Hello");
}

Since the condition is false, the loop will never be entered and no output will be shown. However, compare that to this do while loop:

do {
    System.out.println("Hello");
} while (false); // only checks the condition after running once

Since the inner code executes once before checking the condition, we will get Hello output to the console.
