For Loops

Back to Table of Contents

The first type of loop we will cover is the for loop. The general syntax of this loop is as follows:

for (setup ; condition ; incrementer ) {
	// Your code here
}

There are three main parts to a for loop:

  1. The setup: This statement is run only once at the very beginning of the loop and is usually used to initialize the counter of the loop.
  2. The condition: When the condition evaluates to false, the loop will terminate.
  3. The incrementer: The last part is the statement which increments the counter variable from the initialization. This statement is called each time the loop finishes executing its inner code.

    4. 
			
  4. The loop then checks if the condition i <= 3 is still true. If so it will execute the inner code again. Since 2 <= 3 is true, it will execute the inner code.
    5. 
			
  5. The steps repeat from step 2 again, this time printing out 2 since i was incremented in step 3.
    6. 
			
  6. After the third run through, i++ makes i become 4. When the condition is checked, 4 <= 3 is no longer true, and the loop will terminate, preventing it from printing out 4.
