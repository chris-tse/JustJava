For Loops

Back to Table of Contents

The first type of loop we will cover is the for loop. The general syntax of this loop is as follows:

for (initialization ; condition ; incrementer ) {
	// Your code here
}

There are three main parts to a for loop:

  1. The initialization: This statement is run only once at the very beginning of the loop and is usually used to initialize the counter of the loop.
  2. The condition: This is an expression which evaluates to a boolean which while it is true the loop will continue running.
  3. The incrementer: The last part is the statement which increments the counter variable from the initialization. This statement is called each time the loop finishes executing its inner code.
