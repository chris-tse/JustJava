Programs written in Java are always executed line by line from top to bottom (with a few certain exceptions). But what happens if sometimes we want code to run depending on a condition?

In order to do that, we have statements in Java such as if and else which we can use to direct the flow of code execution. We finished Chapter 1 with the boolean type, so let's take a look at how we use true and false in programming.
