While Loops

Back to Table of Contents

The while loop works the same way as a for loop, but with slight changes:

while (condition) {
	// Your code here
}

Again, the inner code will continue to execute while the condition is true and will terminate when it becomes false. However, there is no setup or incrementer for the counter variable. This means that we will have to initialize it outside the loop and increment it from within the loop.

So why are there different ways of writing loops when they accomplish the same thing? 


		
A for loop has an setup and incrementer, and so it makes more sense to use it when you want to do something for a specific number of times. The while loop is a bit more open on what the condition checks, and the condition may not always depend on the number of times the loop is run.
