While Loops

Back to Table of Contents

The while loop works the same way as a for loop, but with slight changes:

while (condition) {
	// Your code here
}

Again, the inner code will continue to execute while the condition is true and will terminate when it becomes false. However, there is no setup or incrementer for the counter variable. This means that we will have to initialize it outside the loop and increment it from within the loop.

So why 

		
    
			
  1. The loop runs the setup part, initializing the counter variable int i = 1;.
    2. 
			
  2. The code inside the loop executes, printing out i which is 1.
    3. 
			
  3. Now that the inner code is all done, the incrementer will be called, i++ which increments i to 2.
    4. 
			
  4. The loop then checks if the condition i <= 3 is still true. If so it will execute the inner code again. Since 2 <= 3 is true, it will execute the inner code.
    5. 
			
  5. Repeat steps 2 - 4, updating the value of i as it gets incremented.
    6. 
			
  6. After the third run through, i++ makes i become 4. When the condition is checked, 4 <= 3 is no longer true, and the loop will terminate, preventing it from printing out 4.
