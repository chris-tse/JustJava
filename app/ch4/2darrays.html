2-D Arrays

Back to Table of Contents

Arrays as we have used them up until now have been only values in a single row; single dimension. However, it is possible to store values in a two dimensional manner using arrays as well, and that is 2-D arrays.

Generally, a 2-D array consists of an array which holds several other arrays in order to create a rectangular grid shape. This is helpful for keeping track of data that is arranged in a grid-like form, like a game of tic-tac-toe for example.

Let's say we wanted to represent the state of this game of tic-tac-toe in a program:



		
The output we'll obtain from this would be:


		
2 0 1
1 1 2
2 0 0


		
As can be seen in the example above, a 2-D array is nothing more than an array of arrays. However, this does make it a little more complex when we want to go through the 2-D array using loops since we now have to go through two different values: the index of the main array, and the index of each of the inner arrays.

		

		
In the example above, to print the values of the array we use something called a nested loop, which means that there is one loop running within another.


		
Reading Values


		
In the previous section, we printed out the array by calling System.out.println() on each element, but there are easier ways. One of them is by using a loop like above:


		
for (int i = 0; i < evens.length; i++) {
	System.out.println(evens[i]);
}



		
There is also something called a for each loop, also known as enhanced loops which is more concise:


		
for (int num : evens) {
	System.out.println(evens[i]);
}


		
You will notice that it does not have the usual elements of a for loop. With this type of loop, it is the equivalent of saying "for each num in the array evens execute the following code block, hence the name for each.


		
Performing Calculations


		
Using this looping technique, we can perform tasks using the values inside the array. If we use the example from the previous section with the bookshelf, we could calculate the total number of books using the number of books on each row.


		
int[] booksOnRow = {16, 19, 20, 18, 15};

int totalBooks = 0; // initialize variable to hold number while summing
for (int i = 0; i < arr.length; i++) {
	totalBooks += booksOnRow[i]; // add the number of books 
	                             // of each row to the running sum
}

System.out.println(totalBooks);
