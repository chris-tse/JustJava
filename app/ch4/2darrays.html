2-D Arrays

Back to Table of Contents

Arrays as we have used them up until now have been only values in a single row; single dimension. However, it is possible to store values in a two dimensional manner using arrays as well, and that is 2-D arrays.

Generally, a 2-D array consists of an array which holds several other arrays in order to create a rectangular grid shape. This is helpful for keeping track of data that is arranged in a grid-like form, like a game of tic-tac-toe for example.

Let's say we wanted to represent the state of this game of tic-tac-toe in a program:



		
The output we'll obtain from this would be:


		
2 0 1
1 1 2
2 0 0


		
As can be seen in the example above, a 2-D array is nothing more than an array of arrays. However, this does make it a little more complex when we want to go through the 2-D array using loops since we now have to go through two different values: the index of the main array, and the index of each of the inner arrays.

		

		
In the example above, to print the values of the array we use something called a nested loop, which means that there is one loop running within another.
