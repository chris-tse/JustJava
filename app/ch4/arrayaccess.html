Array Access and Modification

Back to Table of Contents

When writing programs, we need ways to manage and store data efficiently, accurately, and concisely. To do this, we use a wide variety of data structures. In this chapter we will be covering arrays, which is one of the most basic data structures, but has a wide range of uses.

Here is the general structure of defining a basic array:

type[] variableName = {/* Comma separated values */};

Once the type of the array has been defined, that array can only hold values of the same type.

An array allows us to store multiple pieces of data in one place like a list. For example, let's say we have a bookshelf with 5 rows. There are 16 books on the first row, 19 books on the second row, 20 on the third, 18 on the fourth, and 15 on the last row. If we wanted to keep track of the number of books in each row, we might do something like this:

int booksOnRow1 = 16;
int booksOnRow2 = 19;
int booksOnRow3 = 20;
int booksOnRow4 = 18;
int booksOnRow5 = 15;

While this might work fine for a smaller program, what if we need to define a larger number of rows, or multiple bookshelves? We could modify the variable names with more numbers to keep track of which belongs where, but it gets messy very quickly.

What if we could store number of books on the bookshelf as a whole in one place? With arrays this is possible. Here's how you might define the same bookshelf from above with an array:

int[] booksOnRow = {16, 19, 20, 18, 15};
