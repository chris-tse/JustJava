Now that we know how to create arrays, retrieve values, and change values, we can use loops to manipulate them more efficiently.

Setting Values

To set or change the values in a loop, it is easier to set them using a loop if the values are calculated in a certain way. For example, if we wanted an array of 5 even numbers starting from 0:

int[] evens = new int[5];

// goes through each index, setting that value to be double its index
// e.g. 0 -> 0
//      1 -> 2
//   	2 -> 4
for (int i = 0; i < evens.length; i++) {
	evens[i] = i * 2;
}

// Result: evens = {0, 2, 4, 6, 8}

By using evens.length instead of just 5 in the condition, we can simply change the length in the first line and it will continue working without additional changes.

Reading Values

In the previous section, we printed out the array by called System.out.println() on each element, but there are easier ways. One of them is by using a loop like above:

for (int i = 0; i < evens.length; i++) {
	System.out.println(evens[i]);
}

There is also something called a for each loop, also known as enhanced loops which is more concise:

for (int num : evens) {
	System.out.println(evens[i]);
}

You will notice that it does not have the usual elements of a for loop. With this type of loop, it is the equivalent of saying "for each num in the array evens execute the following code block, hence the name for each.

Performing Calculations

Using this looping technique, we can perform tasks using the values inside the array. If we use the example from the previous section with the bookshelf, we could calculate the total number of books using the number of books on each row.

int[] booksOnRow = {16, 19, 20, 18, 15};

int totalBooks = 0; // initialize variable to hold number while summing
for (int i = 0; i < arr.length; i++) {
	totalBooks += booksOnRow[i]; // add the number of books 
	                             // of each row to the running sum
}

System.out.println(totalBooks);
