Now that we know how to create arrays, we can retrieve the values in them for other tasks or modify them to update the data they store.

Reading from Arrays

To access the individual values of an array, we use the square bracket notation with the index of the array in this manner:

int[] arr = {1, 2, 3};

// Indices of arrays in Java start at 0
System.out.println(arr[0]); // prints first position: 1
System.out.println(arr[1]); // prints second position: 2
System.out.println(arr[2]); // prints third position: 3

We need to remember that in Java, the program counts the positions starting from 0, not 1. Since we covered loops in the previous chapter, we can print all the values of an array much more easily:

for (int i = 0; i < arr.length; i++) { // the length of an array can be called with the .length property
	System.out.println(arr[i]);
}

Since the position starts from 0, the loop should start at 0 and go until 1 less than the length of the array (i.e. last position of array of length x is index x-1).

Using this looping technique, we can perform tasks using the values inside the array. If we use the example from the previous section with the bookshelf, we could calculate the total number of books using the number of books on each row.

int[] booksOnRow = {16, 19, 20, 18, 15};

int totalBooks = 0; // initialize variable to hold number while summing
for (int i = 0; i < arr.length; i++) {
	totalBooks += booksOnRow[i]; // add the number of books 
	                             // of each row to the running sum
}

System.out.println(totalBooks);

Changing Values in Arrays

While this might work fine for a smaller program, what if we need to define a larger number of rows, or multiple bookshelves? We could modify the variable names with more numbers to keep track of which belongs where, but it gets messy very quickly.

What if we could store number of books on the bookshelf as a whole in one place? With arrays this is possible. Here's how you might define the same bookshelf from above with an array:

int[] booksOnRow = {16, 19, 20, 18, 15};
