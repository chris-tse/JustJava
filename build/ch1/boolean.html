Boolean Type

In Java, we also have a data type used to store whether something is true or false. Java stores these values in the boolean data type, which only hold either true or false

In order to check whether something is true or false, we use equality and relational operators. They are as follows:

  • ==    equal to
  • !=    not equal to
  • >    greater than
  • >=    greater than or equal to
  • <    less than
  • <=    less than or equal to

When declaring a character in Java, we use the type char and surround the character with single quotes.

For several characters in a row, like whole words and sentences, a String type must be used, and the string should be wrapped in double quotes (note that String is capitalized).
