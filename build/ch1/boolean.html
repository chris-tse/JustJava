Boolean Type

In Java, we also have a data type used to store whether something is true or false. Java stores these values in the boolean data type, which only hold either true or false. We can assign them to variables like any other value:

boolean isStudent = true; // is a student
boolean isHungry = false; // is not hungry

We can also invert the value using the ! symbol, known as the unary invert operator, which simply gives the opposite value. For example:

boolean b = true;
System.out.println(!b); // prints out false
