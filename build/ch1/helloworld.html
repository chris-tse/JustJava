As with learning any programming language, we'll start off with a "Hello World" program so you can see what a program written in Java might look like. 



		
Running that bit of code will print out the following:


		
Hello World!


		
There is quite a lot involved here, but we will only focus on these two lines for now:


		
public static void main (String[] args) {}


		
This is called the main method, and it is usually run first when the program starts. And inside this, we have the actual code which we want to run:


		
System.out.println("Hello World!");


		
Without going into too much detail, System.out.println() is a method which prints out whatever is inside the parentheses and also outputs a newline.


		
If we get into detail, this will get complicated very quickly, but for now, practice programs can be written by copying the example below and changing the code within the main method, changing "HelloWorld" on the first line to the same as your filename.
