Hello World

As with learning any programming language, we'll start off with a "Hello World" program so you can see what a program written in Java might look like. There might be quite a bit going on at once, and we'll cover them in more detail later on, but we'll briefly go over this example program.



		
Running that bit of code will print the following to either your terminal window if you're running it from the command line, or the built in console window if you're using an IDE:


		
Hello World!


		
Quite a bit of code just to print out one line, huh? It's actually not so bad once we break it up. We really only need to focus on two lines.


		
public static void main (String[] args) {}


		
This is called the main method, and for now, all we need to know is that whenever a Java program runs, it will usually run whatever is inside this block first. And inside this, we have the actual code which we want to run:


		
System.out.println("Hello World!");


		
Without going into too much detail, System.out.println() is a method which prints out whatever is inside the parentheses and also outputs a newline.


		
If we get into detail, this will get complicated very quickly, but for now, practice programs can be written by copying the example below and changing the code within the main method.
