AND and OR

Back to Table of Contents

Now that we've looked at ways to compare two values, what if we need to make a decision based on whether two or more things are true or false? To do this we use two logical operators called the AND and OR operators, represented by && and || respectively.

Here is a table which shows the result of these operators depending on x and y



		
These correspond to how we use these terms in the English language as well. consider the follow statements here:


		
boolean c = a && b; // a and b are boolean types
boolean d = a || b;


		
In English, we'd say that c is true when both a AND b are true, otherwise it is false. Similarily, we can say that d is true when either one of a OR b is true.


		
Why do we need these operators? We will see the use of these in the next section when we cover if statements. These statements rely only on one value of true or false, so if you have multiple conditions to check, you will either need use many if statements, or have a way to represent many conditions as a single value using logical operators.
