else and else if statements

Back to Table of Contents

With if statements we can make certain decisions in our program based on certain conditions. If we look at the practice problem from the previous section, we checked if (grade >= 50) and if (grade < 50) to determine is it was a passing or failing grade. However, this creates extra unnecessary work for the program since if either one is false, it implies the other.

We can resolve this issue by using else statements. The general structure of an if/else statement is as follows:

if (condition) {
	// Executes if condition was true and ignores the else block
} else {
	// Executes if condition was false and ignores the block above		
}

By using the else statement we can save the program from having to evaluate a second condition. While this may not seem like much, when you have many conditions to check throughout your program they begin to add up. Furthermore, the if/else structure makes more logical sense and makes reading the code a lot easier.

The example above involves only two states: 21 and above, or below. If we have a situation where there is more than two possible states of a condition, we can chain another if statement after each else.


		
if (condition1) {
	// Executes if condition1 is true and skips the rest
} else if (condition2) { 		
	// Executes if condition2 is true and skips the rest
} else {
	// Executes if none of the above conditions were true		
}


		
You can continue chaining as many else if statements as needed. With lots of conditions to check, it may look messy, and we'll cover a different way to check several conditions using the switch statement.
