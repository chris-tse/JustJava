Array Basics

Back to Table of Contents

When writing programs, we need ways to manage and store data efficiently, accurately, and concisely. To do this, we use a wide variety of data structures. In this chapter we will be covering arrays, which is one of the most basic data structures, but has a wide range of uses.

Here is the general structure of defining a basic array:

type[] variableName = new type[length];

This will initialize an array with length values all set to 0. Once the type of the array has been defined, that array can only hold values of the same type and the length cannot be changed.

An array allows us to store multiple pieces of data in one place like a list. For example, let's say we have a bookshelf with 5 rows. There are 16 books on the first row, 19 books on the second row, 20 on the third, 18 on the fourth, and 15 on the last row. If we wanted to keep track of the number of books in each row, we might do something like this:

int booksOnRow1 = 16;
int booksOnRow2 = 19;
int booksOnRow3 = 20;
int booksOnRow4 = 18;
int booksOnRow5 = 15;

While this might work fine for a smaller program, what if we need to define a larger number of rows, or multiple bookshelves? We could modify the variable names with more numbers to keep track of which belongs where, but it gets messy very quickly.

What if we could store number of books on the bookshelf as a whole in one place? With arrays this is possible. Here's how you might define the same bookshelf from above with an array:

// initialize an array of integers of length 5
int[] booksOnRow = new int[5];

booksOnRow[0] = 16; // set first position to 16
booksOnRow[1] = 19; // set second position to 19
booksOnRow[2] = 20; // set third position to 20
booksOnRow[3] = 18; // set fourth position to 18
booksOnRow[4] = 15; // set fifth position to 15

We use indices to set individual values of an array. Note that the count of the position starts at 0 and not 1. We access the values using square brackets and the index of the value, and we can then treat it like a regular int variable.

While this seems like just as much work as before, all the values are contained neatly within one variable rather than being spread out across dozens of individual variables. An example would be to make several arrays and name them according to their book genre or topic.

fantasy[0]; // books in first row of fantasy shelf
mystery[2]; // books in third row of mystery shelf

If we know all the values we want to set, we can use this shorthand way of initializing an array:

int[] booksOnRow = {16, 19, 20, 18, 15};
