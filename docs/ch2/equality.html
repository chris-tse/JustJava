At the end of Chapter 1 we saw that there is a boolean data type which can be either true or false, but by themselves they are not very useful.

A majority of the time, booleans come into play when we want to compare data. We can compare the primitive data types covered in Chapter 1 (except String) using the following operators:

  • ==    equal to
  • !=    not equal to
  • >    greater than
  • >=    greater than or equal to
  • <    less than
  • <=    less than or equal to

These are the same concepts that you may have learned in math class. Here is how we use them:

// a will evaluate to true, since 3 is less than 4
boolean a = 3 < 4;
// b will evaluate to false since 4 is not greater than 150.19
boolean b = 4.0 > 150.19
// c will evaluate to true since true is not false
boolean c = true != false
