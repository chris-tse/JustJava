Ternary Operator

Back to Table of Contents

In computer science, a ternary operator is an operator that takes three arguments in a single line of code, the Java ternary operator let's you assign a value to a variable based on a boolean expression â€” either a boolean field, or a statement that evaluates to a boolean result.

The syntax of ternary operator:

    condition ? expre1 : expr2;

If condition is true, the operator returns the value of expr1; otherwise, it returns the value of expr2. For example, if you want to compair the value a and b, and return small value, you can do this:

a > b? b : a ;

The condition statement is a > b , if a > b is true, it will return expression 1 which is b, if the condition is false, it will return the expression 2 which is a.

Below are some examples that demonstrating some basic example using ternary operator 

// if (a+b) great than c, return a, else return b
int result = (a+b) > c ? a : b;
//if a is not equal to b, print a, otherwise print b
(a != b) ? System.out.print(a): System.out.print(b);
